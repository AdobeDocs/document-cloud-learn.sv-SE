# Guidade åtgärder

För att kunna leverera de bästa kundupplevelserna måste innehållet skapas snabbare än någonsin tidigare. Lär dig använda funktionsmakron för att köra ett eller flera kommandon i flera dokument med ett enda klick så att du sparar tid och tangenttryckningar. Den här videosjälvstudiekursen använder den [nya Acrobat-upplevelsen](../getting-started/new-experience.md).
